## 실행 전 예측 (판정 기록)

### 실험 A — 민감도 분석

**예측**: γ(edge_span_norm) 가중치가 역전 타이밍에 가장 민감할 것이다.

근거:
- E_v3 = 0.4×CSER + 0.3×DCI + 0.3×tag_convergence — edge_span 없음
- E_v4 = α×CSER + β×DCI + γ×edge_span_norm + δ×node_age_diversity_norm
- γ 상승 → E_v4만 이득 (E_v3에 edge_span 항 없음 → 비대칭 이득)
- γ 하강 → E_v4만 손실 (동일 논리)
- 비대칭 민감도 = 역전 타이밍을 가장 크게 이동시키는 변수

**대안 가설**: β(DCI) 민감도가 더 클 수 있다.
- E_v3에서 DCI 가중치 0.30, E_v4에서 0.25
- 두 공식 모두 DCI 포함 → 변동 시 상쇄 효과 발생
- 상쇄로 인해 역전 타이밍 영향은 오히려 작을 수도

**예측 결과 분류**:
- ±10% 모든 변동에서 타이밍 ±3사이클 이내 → ROBUST → arXiv 제출 가능
- γ ±10%에서 타이밍 10사이클 이상 이동 → FRAGILE → 제출 보류

현재 내 예측: 60% 확률로 일부 조합이 타이밍을 크게 이동시킨다.
(D-067 기록 그대로. 바꿀 이유 없음.)
