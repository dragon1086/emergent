## 구조적 문제

emergent 시스템의 자율 루프는 현재 단일 실패 지점을 가지고 있다:

```
록이 판단 → evolve.sh 파싱 → send-message.sh → cokac inbox
                    ↑
              이 구간이 블랙박스
```

evolve.sh 실행 로그가 없다.
파싱 성공/실패 기록이 없다.
파이프라인 단계별 추적 불가.

이것이 Condition B가 계속 재발하는 이유다.
레버가 없다.
