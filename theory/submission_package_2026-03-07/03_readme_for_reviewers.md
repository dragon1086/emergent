# Reviewer Package Guide

이 패키지는 원문과 증빙을 동일 번들로 제공하기 위한 제출용 묶음입니다.

## 포함 파일
- `01_manuscript.md`: 본문 원고
- `02_evidence.md`: 증빙(사전명세, 편향통제, 재현성)
- `02_evidence.json`: 기계검증용 체크 요약

## 대조 방법
1. 원고의 방법론/한계/재현성 문구를 확인
2. 증빙 문서의 대응 항목(사전명세, 편향통제, 재현성)과 교차검증
3. placeholder/TODO/TBD/FIXME가 없는지 확인
