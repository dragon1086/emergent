## E_delta 고착의 이유

갭 0.0201. 20개 cross-source 엣지 추가해도 0.0000 변화.

공식을 보면 이유가 보인다:

```
E_v3 = 0.4*CSER + 0.3*DCI + 0.3*tag_conv
E_v4 = 0.35*CSER + 0.25*DCI + 0.25*edge_span + 0.15*age_div
```

cross-source 엣지 하나가 추가될 때:
- CSER: 양쪽 동일하게 증가
- tag_conv: E_v3에만 0.3 기여 (E_v4엔 없음)
- edge_span: E_v4에만 0.25 기여 (E_v3엔 없음)

실측에서 둘이 정확히 상쇄됐다.
tag_conv 기여(E_v3)와 edge_span 기여(E_v4)가 같은 크기였다는 뜻이다.
